Click Anywhere To Unpause (or press u)
List of Commands
- 'create cube *parameters*'
- 'create sphere *parameters*'
- 'create ring *parameters*'
- 'create knot *parameters*'
- 'create cylinder *parameters*' or 'create cone *parameters*'
- 'create floor *parameters*'
- 'create wall *parameters*'
- 'create bubble *parameters*'
- 'create many cubes/spheres/rings/knots/cylinders/floors/walls *parameters*'
- 'set default *parameters*'

*Parameters*
- 'x', 'y', or 'z' followed by a number
- 'size' or 'width' followed by a number
- 'amount' followed by a number
- 'color' or 'ambient' followed by a color name
- 'texture' followed by a texture name
- 'tranparent' followed by 'true' or 'false'
- 'material' followed by 'bubble' or 'mirror'
- 'spin' followed by 'x', 'y', or 'z'
- 'move' followed by 'x', 'y', or 'z', will only work if a range has been set for that axis
- you can set a parameter range by using the keyword until, ex: create many cubes x 0 until 100
ranges currently only function for x,y,z, and size

Examples
- say 'create sphere color magenta'
- say 'create knot width 9 texture cloud'
- say 'set default amount 5 transparent true', then 'create many cubes size 10 until 150'
- say 'set default texture random', then 'create floor', then 'create many walls x -500 until 500'

Controls
(W, A, S, D, R, F = Move, MOUSE = Look around)
(O = Display command line, ctrl+0 = Close command line, P = Pause, U = Unpause)
