Click Anywhere To Unpause (or press u)
Examples
- say 'create cube x 75 y 100 color magenta'
- say 'create floor', then 'create knot material mirror spin y slow'
- say 'set default amount 5 transparent true', then 'create many cubes size 10 until 150'
- say 'set default texture random', then 'create floor', then 'create many walls x -500 until 500'
- say 'set default x -50 until 50 y 0 until 100', then 'create many cones move x spin z'

List of Commands
- create geometry, followed by optional parameters
- create many geometries, followed by optional parameters
- set default, followed by parameters
- reset default, open/close console, gravity on/off, undo, save, upload, test

Geometry Names
cube - sphere - cylinder - cone - ring - knot - floor - wall - back wall - text

Parameters
everything in 'single quotes' represents the exact words you say
everything in (parentheses) represents your input
everything in [square brackets] is optional
the symbol -> represents followed by
- 'x', 'y', or 'z' -> (number) [ -> 'until' -> (number) ]
- 'size', 'width', 'height', or 'depth' -> (number)
- 'color' or 'ambient' -> (color name)
- 'texture' -> (texture name)
- 'material' -> (material name), right now the only options are 'bubble' or 'mirror'
- 'tranparent' -> 'true' or 'false'
- 'amount' -> (number), this parameter is used when creating many geometries
- 'spin' -> 'x', 'y', or 'z' [ -> 'slow', 'medium', or 'fast' ]
- 'move' -> 'x', 'y', or 'z' [ -> 'slow', 'medium', or 'fast' ]
- **move will only work if a range has been set for that axis using the 'until' keyword
- 'solid' -> 'true' or 'false', this makes a geometry collidable when gravity is on

Controls
(W, A, S, D, R, F = Move, MOUSE = Look around)
(O = Open console, ctrl+0 = Close console, H = Hide/show gui, P = Pause, U = Unpause)
